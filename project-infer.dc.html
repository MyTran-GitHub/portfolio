# Modular Backend Implementation in CausalImagesR
class SatelliteEngine:
    def __init__(self, backend="alpha_earth"):
        # Swappable backends: ["cnn", "vit", "clay", "alpha_earth"]
        self.extractor = load_model(backend)

    def get_matching_pixel(self, target_image, pool):
        # Top-K Cosine Similarity Retrieval
        embeddings = self.extractor.encode(pool)
        target_vec = self.extractor.encode(target_image)

        return get_top_k_neighbors(target_vec, embeddings, k=50)

How to read the result

Low-intensity prescribed fire reduces subsequent high-intensity wildfire risk by 4.4% over a six-year horizon (2012–2018), with effects concentrated in embedding-matched control pixels that show the strongest pre-treatment balance.

Small number, large stakes — at California's scale, a few percentage points of avoided catastrophic fire is measured in communities, not pixels.
